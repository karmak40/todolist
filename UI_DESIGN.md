## Элементы интерфейса в деталях

### Заголовок с профилем
```
┌──────────────────────────────┐
│ ┌────┐                       │
│ │ 👤 │  Доброе утро         │
│ │    │  Пользователь        │
```

- Радиус: 30dp
- Иконка по умолчанию: 👤

## Диалог добавления задачи

```
┌──────────────────────────────────┐
│  Добавить новую задачу     [×]    │
├──────────────────────────────────┤
│ Название:                        │
│ [Введите название задачи       ]│ ← обязательное
│                                  │
│ Описание:                        │
│ [Введите описание        ]      │
│ [продолжение           ]        │
│                                  │
│ Длительность (минут):           │
│ [30                           ]│
│                                  │
│ Дата:                           │
│ [22/02/2026          ]  [📅]    │ ← DatePicker
│                                  │
├──────────────────────────────────┤
│ [Отмена]        [Добавить]       │
└──────────────────────────────────┘

Ширина диалога: match_parent (-48dp)
Максимальная ширина: 400dp

TextField:
- Padding: 16px
- Border: OutlineInputBorder
- Hint text: gray

Кнопки:
- TextButton для отмены
- ElevatedButton для добавления
```

## Адаптивность

### Портретная ориентация (готовая)
```
┌─────────────────────────────┐
│   Полный экран              │
│   Все элементы видны        │
│   Скролируемые списки        │
└─────────────────────────────┘
```

### Альбомная ориентация (база готова)
```
Для расширения:
- Увеличить размер карточек
- Двух-колоночный список
- Боковая панель
```

### Планшет (база готова)
Для расширения:
- Master-detail view
- Больше информации на экране
- Более крупные элементы
